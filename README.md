# CangreBot

Bot de la comunidad de Discord de RustLang en Español.

## Desarrollo

```bash
# cargo-watch (optional for dev enviroment)
cargo watch -c -x 'run --bin cangrebot'

# Or simply
cargo run --bin cangrebot
```

## Producción

```bash
cargo build --release --bin cangrebot
```
